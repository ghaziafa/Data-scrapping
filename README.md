#Data Scraping & Web Data Collection
### Daten-Scraping & Web-Datenerfassung

---

##English

### Project Overview
This repository contains two end-to-end data scraping and cleaning projects that demonstrate the ability to collect data from online sources, clean and process it, and prepare it for business intelligence reporting and analysis.

---

###Projects in this Repository

---

#### 1. Online Data Scraping for Business Intelligence
**File:** `OnlineDataScraping4BI.ipynb`

**Description:**
A complete data scraping pipeline that fetches data from online sources and transforms it into a structured format ready for Business Intelligence (BI) tools such as Power BI or Tableau.

**What it covers:**
- Sending HTTP requests to fetch web data using `requests`
- Parsing HTML content using `BeautifulSoup`
- Structuring raw scraped data into pandas DataFrames
- Cleaning and transforming data for BI-ready output
- Exporting clean data for dashboarding and reporting

**Tools used:**
| Tool | Purpose |
|------|---------|
| Python | Core scripting language |
| requests | HTTP requests to fetch web pages |
| BeautifulSoup | HTML parsing and data extraction |
| pandas | Data structuring and cleaning |
| Jupyter Notebook | Development environment |

---

#### 2. Web-Scraped E-commerce Data Cleaning
**File:** `Web-Scraped E-commerce Data Cleaning.ipynb`

**Description:**
A data cleaning project focused on raw e-commerce data collected from the web. The project demonstrates how to handle messy, real-world scraped data and prepare it for analysis and reporting.

**What it covers:**
- Handling missing values and null entries
- Removing duplicates and irrelevant records
- Standardising text formatting and data types
- Cleaning price, rating and category fields
- Preparing a clean, analysis-ready dataset

**Tools used:**
| Tool | Purpose |
|------|---------|
| Python | Core scripting language |
| pandas | Data cleaning and transformation |
| NumPy | Numerical operations |
| Jupyter Notebook | Development environment |

---

### Key Skills Demonstrated
- Web scraping using `requests` and `BeautifulSoup`
- Data pipeline construction from raw source to clean output
- Real-world data cleaning techniques on messy scraped data
- Preparing datasets for Business Intelligence tools
- Exploratory data analysis on e-commerce data

---

### How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/ghaziafa/Data-scrapping.git
   ```
2. Install dependencies:
   ```bash
   pip install pandas numpy requests beautifulsoup4
   ```
3. Open either notebook in Jupyter and run all cells from top to bottom
